## 1. 需求背景/概述

### 1.1 项目背景

兆涵科技 Sigma Education AI教学助手是一款面向中小学教师的智能文档生成工具，旨在帮助教师快速生成高质量的教学工作计划、校园活动策划方案和家长会发言稿。

当前教师在日常工作中需要频繁撰写各类教学文档，传统方式费时费力。本产品通过AI技术辅助教师高效完成文档生成工作，提升工作效率。

### 1.2 产品定位

面向中小学教师群体的AI智能写作辅助工具，专注于教学场景下的文档生成需求。

### 1.3 核心价值

- **高效便捷**：通过简单的关键词输入，快速生成完整文档
- **模板丰富**：提供多种精选模板，覆盖常见教学场景
- **易于编辑**：支持在线编辑和二次修改
- **历史复用**：自动保存生成历史，方便重复使用

---

## 2. 产品目标

### 2.1 核心目标

1. 帮助教师在3分钟内完成一份完整的教学工作计划、活动策划或家长会发言稿
2. 提供3种核心文档类型的智能生成能力
3. 建立可持续扩充的模板库系统

### 2.2 功能目标

| 功能模块 | 目标描述 |
|---------|---------|
| 文档生成 | 支持3种文档类型的模板填充式生成 |
| 历史管理 | 支持查看、编辑、删除、重生成历史记录 |
| 模板中心 | 提供精选模板库，支持分类浏览和搜索 |
| 文档编辑 | 支持在线编辑、下载为txt文件 |
| 结果展示 | 在独立窗口展示生成结果，支持重新生成 |

### 2.3 非功能目标

- 页面加载时间 < 2秒
- 生成响应时间 < 3秒（模拟，实际依赖AI接口）
- 支持主流浏览器（Chrome、Firefox、Safari、Edge）
- 响应式布局，支持桌面和移动端访问

##### 3.2.4.4 结果页通用功能

**编辑功能详细规则：**
1. 点击"编辑"按钮
2. 内容区域切换为textarea，显示格式化文本
3. HTML标签被解析为纯文本显示
4. 按钮切换为"确认"和"取消"
5. 点击"确认"：将编辑内容转换回HTML保存
6. 点击"取消"：恢复原始内容

**编辑文本转换规则：**
```
输入 → 输出
-------------------------
段落文本 → <p>文本</p>
• 开头的行 → <li>文本</li>（连续多个自动包裹<ul>）
一、/第X节/【】开头的行 → <h2>文本</h2>
空行 → 段落分隔
```

**下载功能：**
- 生成带BOM的UTF-8编码txt文件
- 文件名格式：[文档标题]_[日期].txt
- 示例：小学三年级语文教学工作计划_20260527.txt

**重新生成：**
- 返回首页
- 自动填充上次输入内容到输入框
- 保留选中的文档类型标签
